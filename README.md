# Flipkart Sales Dashboard

## Overview
This project features a comprehensive Power BI dashboard that analyzes sales data from Flipkart. It provides interactive visualizations and insights into sales performance, customer trends, and product metrics.

## Features
- **Interactive Visualizations:** dynamic charts and graphs to explore data.
- **Sales Insights:** Analysis of revenue, sales volume, and categorical performance.
- **Data Source:** Based on `flipkart_sales_data.xlsx`.

## Files Included
- **`flipkart dashboard.pbix`**: The main Power BI report file containing the dashboard and data model.
- **`flipkart_sales_data.xlsx`**: The Excel dataset used as the source for the dashboard.
- **Images**: Logo files (`flipkart logo.png`, etc.) used within the report or for branding.

## Prerequisites
- **Microsoft Power BI Desktop**: You need to have Power BI Desktop installed to view and interact with the `.pbix` file. [Download here](https://powerbi.microsoft.com/en-us/desktop/).

## How to Use
1.  **Clone or Download** the repository.
2.  **Open** `flipkart dashboard.pbix` using Microsoft Power BI Desktop.
3.  **Interact** with the dashboard by using the filters and slicers to view different data cuts.
4.  If the data needs refreshing, ensure the path to `flipkart_sales_data.xlsx` is correctly set in the Power BI Data Source settings.
